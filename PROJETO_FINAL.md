## 📝 Instruções para Usuários

### Instalação:
1. Execute `ChatPlay Soundboard Setup 1.0.0.exe`
2. Clique em "Next" → "Install"
3. Pronto!

### Primeiro Download do YouTube:
- Aguarde 5-10 segundos (yt-dlp.exe sendo baixado)
- Depois funciona normalmente

- Só instale se quiser microfone virtual
- Download: https://vb-audio.com/Cable/
