REQUIREMENTS:
- VB-CABLE Virtual Audio Device (Optional - for virtual microphone support)
  Download: https://vb-audio.com/Cable/

INSTALLATION:
1. Run ChatPlay Soundboard Setup.exe
2. (Optional) Install VB-CABLE for virtual microphone support
3. On first YouTube download, yt-dlp.exe will be downloaded automatically (~10 MB)

YouTube audio downloads are saved in:
%APPDATA%\chatplay-soundboard\youtube-sounds\
